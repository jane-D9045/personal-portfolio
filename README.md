# Personal Portfolio Website

## Introduction

This project is a Personal Portfolio Website created using HTML5 as part of a Web Development Internship. It provides a simple introduction about the developer, displays technical skills and projects, and includes a contact form for communication.

---

## Project Highlights

- Responsive HTML5 page structure
- About Me section
- Technical Skills section
- Projects section
- Contact Form
- Internal page navigation
- Profile image with alternative text
- Semantic HTML5 elements
- Basic HTML form validation

## Website Structure

The webpage is divided into the following sections:

- Header
- Navigation Menu
- About Me
- Skills
- Projects
- Contact Form
- Footer

Each section is connected through internal navigation links for easy access.

---

## Features Implemented

- Semantic HTML elements (`header`, `nav`, `main`, `section`, `footer`)
- Profile image with `alt` attribute
- Unordered list for skills
- Contact form using Name, Email, and Message fields
- Required field validation
- Email input validation
- Clean and simple webpage layout

---

## How to Run the Project

1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html`.
4. Run the file using Live Server or open it directly in a web browser.
